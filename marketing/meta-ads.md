# Meta 広告 下準備（Instagram / Facebook・β向け・Pixel なし運用）

最終更新: 2026-05-28
ステータス: **下準備のみ**。実配信はオーナー承認・LP 改修完了後のS6以降。

---

## 目的

Instagram / Facebook 経由で LP（https://bati10inter-cmd.github.io/kizuna-baton/lp.html）への流入を増やし、Formspree 経由のウェイトリスト登録を獲得する。

**最終KPI**: ウェイトリスト登録数（Formspree 管理画面で取得）
**中間KPI**: Meta 広告マネージャのリンククリック数・リーチ・インプレッション

---

## 5原則整合（Pixel なし運用）— **最重要**

きずなbaton の5原則「個人情報非取得」と整合させるため、**Meta Pixel / Conversion API は LP に設置しない**。

### 結果として受け入れる制約

| 機能 | Pixel あり | Pixel なし（本方針） |
|---|---|---|
| 配信目的（最適化対象） | コンバージョン最適化 | **リンククリック最適化 or リーチ最適化** のみ |
| リターゲティング | LP 訪問者再アプローチ可 | **不可** |
| ルックアライク（類似オーディエンス） | コンバージョン基盤で生成可 | **不可** |
| CPA レポート | 自動算出 | 手計算（クリック数 × Formspree 登録の時間差相関） |
| 学習効率 | 高い | 低い（CPA は数十%悪化を想定） |

**この制約を受け入れる理由**: 5原則「Cookie/IP 等の個人情報を取得しない」がきずなbaton のコア価値。広告効率より優先する。むしろ「弊サービスは行動追跡しません」を将来 LP のメッセージにできるため、長期的にはブランド資産になる。

## 予算

- **月額**: 5,000〜7,000円（月1万円の総予算のうち Meta 分。Meta CPM が X より高い前提で配分を多めに）
- **日次上限**: 約170〜230円/日
- **配信期間**: 初回は2週間でテスト → 反応を見て継続判断

---

## 配信目的の選択

Meta 広告マネージャでの設定：
- **キャンペーン目的**: 「トラフィック」（リンククリック）
- **最適化**: 「ランディングページビュー」ではなく「**リンククリック**」（ランディングページビューは Pixel必要）
- **入札戦略**: 最低額入札（コスト管理目的）

## クリエイティブ案 3本

X 広告と同じ A/B/C を流用。ただし Meta 用に画像サイズ（1:1 or 4:5）と本文長を調整。

### A. 「家族への一言」訴求（推奨）
- 画像: LP の Family Message Showcase の1カード（Netflix の例）を 1080×1080 に切り出し
- 見出し: 「家族へのひとことごと、契約を整理。」
- 本文（125字以内）:
  > 「もう見る人いなければ解約してね」——サブスクごとに家族への一言を残せる終活アプリ、きずなbaton。事前登録受付中（無料β）。カード番号は預かりません。
- CTA ボタン: 「詳しくはこちら」

### B. 「親の契約整理」訴求
- 画像: Problem セクション「親の契約、整理できていますか？」を 1080×1080 に
- 見出し: 「親の契約整理で、苦労した。」
- 本文:
  > 通帳から落ちる謎の課金、どこに連絡すればいい保険——次世代に同じ思いをさせない、契約の地図。きずなbaton 事前登録受付中。
- CTA ボタン: 「詳しくはこちら」

### C. 「Concept」訴求
- 画像: Phone Mock + Concept 引用
- 見出し: 「保管するのではなく、手渡すアプリ。」
- 本文:
  > カード番号や暗証番号は預かりません。手渡すのは「どんな契約が、どこに、どう解約するか」の地図。Apple Legacy Contact と同じ思想で。
- CTA ボタン: 「詳しくはこちら」

## β段階であることの明示（誤認広告回避）

3案いずれも本文に「**無料β**」または「**事前登録受付中**」を含める。未実装機能（バトンモードの第1/第2段階等）は広告で訴求しない。

## 配信判断フロー

1. LP 改修（S5）完了 ✅
2. クリエイティブ A/B/C を 1080×1080 で書き出し ← S6 で着手
3. オーナー予算承認
4. Meta 広告マネージャでキャンペーン作成（広告アカウント・ビジネスマネージャ未登録なら登録から）
5. 2週間配信 → 中間KPIで A/B/C 比較
6. 反応のあったクリエイティブで継続 or 全停止判断

## 配信中・配信後のチェック項目

- [ ] Meta 管理画面で CTR が0.5%以上か（Instagram は特に高めに出る傾向）
- [ ] LP の Formspree 登録数が配信前比で増加しているか（時間差相関）

- [ ] 5原則違反のフィードバック（Cookie 同意求められない等）が来ていないか
